CloudObject

Cloud framework

How to use CloudObject
CloudObject is a cloud framework. There are two operation include insert and find. Each operation will return in JSON message.

    insert

    It is operation for insertion data. The url below is pattern of insert operation.
    http://www.fbcredibility.com/cloudobject/{user}/insert/{object}?{param1}={param value}&{param2}={param value}
    Insertion operation example of user1
    http://www.fbcredibility.com/cloudobject/user1/insert/student?name=jack&id=5555
    json result
    {"status": 0, "description": "Complete insert"}

    find

    It is operation for finding data which you was add in CloudObject. User can find object by calling at below example url.
    http://www.fbcredibility.com/cloudobject/{user}/find/{object}
    Finding operation example for find student object of user1
    http://www.fbcredibility.com/cloudobject/user1/find/student/
    result
    {"status": 0, "data": [{"id": 5555, "name": "jack"}], "description": "ok"}
