How to use CloudObject
CloudObject is a cloud framework. There are two operation for user include insert and find. Each operation return in JSON message.
    insert
    It is operation for inserting data. The url below is pattern of insert operation.
    http://www.fbcredibility.com/cloudobject/{user}/insert/{object}?{param1}={param value}&{param2}={param value}
    Insert operation example of user test
    find
    It is operation for finding data which you was add in CloudObject. User can find object by calling at below example url.
    http://www.fbcredibility.com/cloudobject/{user}/find/{object}
